# 📚 Library Management System

A web-based Library Management System developed using PHP and MySQL.

## 🚀 Features
- User Login & Registration
- Search Books
- Issue Books
- Renew Books
- Admin Dashboard
- Book Management
- User Management

## 🛠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## 📂 Project Setup
1. Download or clone the repository
2. Import the database into MySQL
3. Configure the database connection
4. Run the project using XAMPP or Laragon
