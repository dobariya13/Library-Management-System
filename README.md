# 📚 Library Management System

A web-based Library Management System developed using PHP and MySQL that helps users search, issue, and renew books efficiently. 

## ✨ Features

- 🔐 User Login & Registration
- 📖 Search Books
- 📚 Issue Books
- 🔄 Renew Books
- 👨‍💼 Admin Dashboard
- 🗂 Book Management
- 👥 User Management
- 📊 Library Record Management

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/parth-dobariyaa/Library-Management-System.git
```

2. Import the database into MySQL.

3. Configure the database connection.

4. Start Apache and MySQL using XAMPP or Laragon.

5. Open the project in your browser.
